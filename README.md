# Overview
This ROS package contains the [fastsim](https://github.com/jbmouret/ros_fastsim)-friendly files necessary to generate a 2-dimensional environment and a robot to reproduce
the experimental setup proposed by J. Tani and S. Nolfi in "Learning to perceive the world as articulated: an approach for 
hierarchical learning in sensory-motor systems".
